## 📋 About This Repository

This repository documents **9 practical networking labs** covering the core topics of the CompTIA Network+ certification. Each lab includes:

- 🎯 **Lab Objectives** — what the lab covers and the Network+ exam objective it maps to
- 🖧 **Network Topology** — real Packet Tracer screenshots of the actual topology built
- 📊 **IP Addressing Table** — all devices with IP addresses, subnet masks and gateways
- ⌨️ **Step-by-Step CLI Commands** — exact Cisco IOS commands used
- ✅ **Verification** — commands to confirm everything works
- 📚 **Key Takeaways** — exam-relevant concepts from each lab
